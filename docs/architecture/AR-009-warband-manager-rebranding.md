# AR-009: Rebrand Application As Warband Manager

## Status

Planned

## Goal

Rename the application from:

Midnight Character Tracker

to:

Warband Manager

## Motivation

The original name was selected during early development.

Since then, the application has evolved beyond a simple tracking tool.

Current functionality includes:

- Character management
- Progress dashboards
- Currency management
- Collection management
- Warband-wide tracking
- User-maintained progression data

The word "Tracker" no longer adequately describes the product.

The term "Midnight" is tied to a specific expansion roadmap and may become outdated over time.

## New Product Name

Warband Manager

## Scope

### Application Branding

Update:

- Window title
- Application title
- About text
- Version information

### Documentation

Update:

- README
- Architecture records
- User guides
- Screenshots
- Development documentation

### User Interface

Update visible references to:

- Midnight Character Tracker

Replace with:

- Warband Manager

### Repository Structure

Evaluate whether repository and package naming should be updated.

This item is optional and may be postponed if migration effort exceeds value.

## Intended Product Identity

Warband Manager shall represent:

- A management application
- A dashboard application
- A collection application
- A progression application

for World of Warcraft Warbands.

## Success Criteria

The application and documentation consistently use:

Warband Manager

as the official product name.
