# Demo Data

This folder contains sample data for evaluating Warband Manager without playing World of Warcraft.

## Option 1 - Populate the Application Quickly

Folder:

character-files/

Copy the files into the application's import folder.

After launching Warband Manager, the characters will be detected automatically.

This is the fastest way to explore the application.

## Option 2 - Test the Import Workflow

Folder:

example-parses/

Open one of the files and copy its contents.

In the application:

Paste Character Data
→ Paste content
→ Import

This demonstrates the normal workflow used when importing data from the WoW addon.
If you skipped option 1 you can use all parses to create characters in the app, in case you
imported chars like described in option 1, the name of the example parses indicate if the parse will
create a new char or if they will update an existing char.

## Included Features

The demo data contains examples of:

- Multiple characters
- Character notes
- Currency tracking
- Warband resources
- Reputation tracking
- Weekly vault progress
- Weekly duties
